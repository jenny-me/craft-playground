# Example of a basic Craft build

This playground site uses a typical base build for Craft including:

**Page Types**

- Home page
- Landing page
- Content page
- Staff listing with detail pages
- Article listing with detail pages

**Plugins**

- CKEditor
- Expanded Singles
- Feed Me
- Field Manager
- Freeform Lite
- Image Resizer
- Retour
- SEOmatic
- Sherlock Lite
- Sprig
